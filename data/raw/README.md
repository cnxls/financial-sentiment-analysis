# Raw Data

## Sentences_50Agree.txt

**Source:** https://github.com/neoyipeng2018/FinancialPhraseBank-v1.0

**Original dataset:** Financial PhraseBank v1.0 by Malo et al. (2014)

**Format:** Each line is `<sentence>@<sentiment>` where sentiment is `positive`, `negative`, or `neutral`.

`Sentences_50Agree.txt` contains sentences where at least 50% of annotators agreed on the sentiment label.
